# Claude Code 세미나: 회의록 정리 자동화

Claude Code는 말을 알아듣는 AI 도구입니다. 코딩 없이, 하고 싶은 작업을 말로 설명하면 실행해줍니다.

이 세미나에서는 매달 쌓이는 회의록을 Claude Code로 자동 정리하는 과정을 직접 체험합니다.

---

## 오늘 실습할 내용

1. 회의록 파일을 깔끔한 업무용 문서로 정리하기
2. "매달 반복된다"고 말 한마디 하면 폴더 구조가 자동으로 생기는 것 보기
3. 정리 방식을 저장해두고, 다음 달 회의록은 짧은 명령 하나로 처리하기

---

## 팀별 실습 폴더

본인 팀에 해당하는 폴더를 선택해 가이드를 따라 진행하세요.

| 폴더 | 대상 |
|------|------|
| `인사팀/` | 채용, 온보딩, 교육, 조직문화 담당 |
| `마케팅_운영팀/` | 마케팅, 캠페인, 물류, 운영 담당 |

각 폴더 안에 `가이드.md` 파일이 있습니다. 그 파일을 열고 순서대로 따라하면 됩니다.

---

## 시작하는 방법

1. 본인 팀 폴더를 Claude Code로 엽니다
2. 폴더 안의 `가이드.md`를 읽습니다
3. 1단계부터 순서대로 진행합니다

---

> 폴더 안에 이름이 이상한 파일(예: `gcp555xxy88.txt`)이 있어도 걱정하지 마세요.
> Claude는 파일 이름이 아니라 내용을 읽어서 어떤 회의록인지 스스로 파악합니다.
