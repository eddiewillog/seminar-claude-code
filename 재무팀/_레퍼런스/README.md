# _레퍼런스 — 진행자용 답안지

> ⚠️ **트레이니에게 미리 보여주지 말 것.**
> 이 폴더는 워크숍 진행자가 트레이니의 결과물과 비교하기 위한 참고 자료다.

## 내용

| 파일 | 비교 대상 |
| --- | --- |
| `CLAUDE.md` | 가이드 7단계에서 트레이니가 만든 `finance-practice/CLAUDE.md` |
| `README_트레이니용_원본.md` | 가이드 8단계에서 트레이니가 만든 `finance-practice/README.md` |
| `requirements.txt` | 트레이니가 만든 환경 안내 |
| `scripts/` | 가이드 2~6단계 동안 클로드코드가 만든 코드와 비교 |

> 이 답안의 `scripts/` 는 원래 `finance-practice/scripts/` 안에 있다고 가정하고 작성됐다.
> 그대로 실행하고 싶다면 `재무팀/finance-practice/scripts/` 로 임시 복사 후 실행한다.
> 보통은 코드 비교 용도로만 읽는다.

## 주의

- 트레이니의 답이 이 레퍼런스와 **완전히 같을 필요는 없다**.
  표준 컬럼이 잡혔고 원본을 손대지 않았고 리포트가 나왔다면 정답이다.
- 클로드코드의 코드 생성 방식은 매번 조금씩 다르다. 결과물의 **품질**을 보고, 코드 자체를 비교하지 말 것.
