## 0단계 — 외부 링크만 던졌을 때 어떻게 되는지 보기

먼저 "그냥 링크 던지면 안 되나?"부터 직접 확인합니다.

아래 내용을 Claude Code에 붙여넣으세요.

```
https://willog.notion.site/PPT-41f9f7bba2994e0cbd368ae24c01e9a8

이 페이지의 브랜드 가이드 참고해서, 같은 폴더의 brief_5월_사내공유회.md 브리프대로
```

**관찰 포인트:** Claude가 페이지 본문을 못 가져오거나 빈손으로 돌아옵니다.

**완료 확인:** "페이지를 불러올 수 없다" 또는 본문 없이 일반론으로 답한 응답을 확인했다면 1단계로 넘어가세요.

## 1단계 — 5월 사내 공유회 PPT 초안 작성 (제대로)

링크 대신 **Claude가 확실히 읽을 수 있는 두 가지 방법** 중 하나를 씁니다.
가능하면 A 먼저 시도하고, 안 되면 B로 가세요.

### 방법 A · 컨플루언스 MCP로 검색해서 읽기

아래 내용을 붙여넣으세요.

```
내 컨플루언스에서 "윌로그 브랜드 리소스 페이지" 를 찾아서 본문을 가져와줘.
그 가이드의 톤·제목·본문·금지 표현 규칙을 적용해서,
같은 폴더의 brief_5월_사내공유회.md 브리프대로 슬라이드 초안을 작성해줘.

형식:
1. 발표 메타 (제목, 대상, 시간, 슬라이드 수)
2. 슬라이드별 구성 (총 10장 기준)
   - 슬라이드 번호 / 제목 / 본문 불릿 / 발표자 노트
3. 브랜드 가이드 준수 체크 (제목 글자수, 한 줄 글자수, 금지 표현 사용 여부)
4. 다음 단계 권고 (디자이너가 직접 손봐야 할 부분)

- 본문은 한 줄 25자 이내, 불릿 3개 이내
- 숫자·고유명사는 **굵게** 표기
- 디자이너가 바로 슬라이드 툴에 옮길 수 있는 수준으로 정리
```

처음 한 번은 Atlassian 로그인이 필요할 수 있습니다. 로그인 안내가 뜨면 그대로 따라하세요.

### 방법 B · 폴더 안 파일 참고하기 (MCP 안 될 때)

아래 내용을 붙여넣으세요.

```
이 폴더에 있는 brand_guide.md 와 brief_5월_사내공유회.md 를 읽어줘.
brand_guide.md 의 톤·제목·본문·금지 표현 규칙을 적용해서,
브리프대로 슬라이드 초안을 작성해줘.

형식:
1. 발표 메타 (제목, 대상, 시간, 슬라이드 수)
2. 슬라이드별 구성 (총 10장 기준)
   - 슬라이드 번호 / 제목 / 본문 불릿 / 발표자 노트
3. 브랜드 가이드 준수 체크 (제목 글자수, 한 줄 글자수, 금지 표현 사용 여부)
4. 다음 단계 권고 (디자이너가 직접 손봐야 할 부분)

- 본문은 한 줄 25자 이내, 불릿 3개 이내
- 숫자·고유명사는 **굵게** 표기
- 디자이너가 바로 슬라이드 툴에 옮길 수 있는 수준으로 정리
```

새로운 문서를 만들 때 "HTML 파일로 작성해줘"라고 요청하면, 크롬 브라우저에서 바로 열어볼 수 있는 형태의 파일로 저장됩니다.

**완료 확인:** 슬라이드 초안 파일이 새로 생겼으면 2단계로 넘어가세요.
