# 디자인팀 실습 가이드 — 브랜드 톤 맞춘 PPT 초안 자동화

이 폴더에는 회사 브랜드 가이드(`brand_guide.md`)와 발표 브리프 파일 2개가 있습니다.
아래 순서대로 진행하세요.

---

## 1단계 — 5월 사내 공유회 PPT 초안 작성

아래 내용을 Claude Code에 붙여넣으세요.

```
이 폴더에 있는 brand_guide.md 와 brief_5월_사내공유회.md 를 읽어줘.
브랜드 가이드의 톤·제목·본문 규칙을 적용해서, 브리프에 정의된 발표용
슬라이드 초안을 아래 형식으로 작성하고 저장해줘.

형식:
1. 발표 메타 (제목, 대상, 시간, 슬라이드 수)
2. 슬라이드별 구성 (총 10장 기준)
   - 슬라이드 번호 / 제목 / 본문 불릿 / 발표자 노트
3. 브랜드 가이드 준수 체크 (제목 글자수, 한 줄 글자수, 금지 표현 사용 여부)
4. 다음 단계 권고 (디자이너가 직접 손봐야 할 부분 안내)

- 슬라이드 제목은 가이드의 제목 규칙(20자 이내)을 지켜줘
- 본문은 한 줄 25자 이내, 불릿 3개 이내
- 숫자·고유명사는 **굵게** 표기
- 가이드 금지 표현("혁신", "최고의" 등)은 자동으로 피해줘
- 디자이너가 바로 슬라이드 툴에 옮길 수 있는 수준으로 정리해줘
```

새로운 문서를 만들 때 "HTML 파일로 작성해줘"라고 요청하면, 크롬 브라우저에서 바로 열어볼 수 있는 형태의 파일로 저장됩니다.

**완료 확인:** 슬라이드 초안 파일이 새로 생겼으면 2단계로 넘어가세요.

---

## 2단계 — 반복 작업용으로 폴더 정리

아래 내용을 붙여넣으세요.

```
이 작업을 발표마다 반복할 예정이야.
폴더를 반복 업무에 맞게 정리해줘.
지금까지 사용한 요청 내용도 함께 저장해줘.
```

**완료 확인:** 폴더 안에 하위 폴더들이 새로 생겼으면 3단계로 넘어가세요.

---

## 3단계 — 작성 방식 저장

아래 내용을 붙여넣으세요.

```
방금 슬라이드 초안을 작성한 방식을 저장해둬.
다음 발표 때 같은 방식으로 바로 사용할 수 있게 brand-deck-draft 라는 이름으로 만들어줘.
```

**완료 확인:** "저장됐습니다" 또는 파일이 생성됐다는 메시지가 나오면 4단계로 넘어가세요.

---

## 4단계 — 6월 파트너 세션 PPT 초안 작성

아래 내용을 붙여넣으세요.

```
/brand-deck-draft

brief_6월_파트너세션.md 의 발표에 대해 5월과 같은 방식으로 슬라이드 초안을 작성해줘.
```

**완료 확인:** 6월 발표용 슬라이드 초안 파일이 새로 생겼으면 완료입니다.

---

## 실습 완료 후

5월·6월 초안을 나란히 열어보세요. **같은 형식·같은 톤**으로 정리되어 있으면 성공입니다.

특히 확인할 점:
- 두 발표 모두 슬라이드 제목이 20자 이내인지
- 본문 글자수 규칙이 지켜졌는지
- 가이드 금지 표현이 들어가지 않았는지

막히는 부분이 있으면 강사에게 문의하세요.
