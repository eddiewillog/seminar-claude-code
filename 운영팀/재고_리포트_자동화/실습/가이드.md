# 주간 재고 리포트 자동화 실습 가이드

이 폴더에는 재고 현황 데이터 파일 1개가 있습니다. 아래 순서대로 진행하세요.

---

## 1단계 — 재고 현황 리포트 작성

아래 내용을 Claude Code에 붙여넣으세요.

```
이 폴더에 있는 재고 현황 파일을 읽어줘.
아래 형식으로 주간 재고 리포트를 작성하고 저장해줘.

형식:
1. 주간 요약 (기준일, 전체 SKU 수, 이번 주 특이사항 한 줄)
2. 긴급 조치 필요 항목 (재고가 10개 이하이거나 이번 주 급격히 감소한 항목)
3. 모니터링 필요 항목 (비고에 경고 표시가 있거나 감소 추세인 항목)
4. 발주 또는 입고 대기 중 항목
5. 점검 대기 중 항목 (가용 전환 예정 포함)
6. 이번 주 조치 권고사항 (담당자 배정 포함)

- 숫자 변화를 보고 추세를 직접 분석해줘
- 비고 내용도 함께 반영해줘
- 실무자가 바로 공유할 수 있는 수준으로 작성해줘
```

새로운 문서를 만들 때 "HTML 파일로 작성해줘"라고 요청하면, 크롬 브라우저에서 바로 열어볼 수 있는 형태의 파일로 저장됩니다.

**완료 확인:** 리포트 파일이 새로 생겼으면 2단계로 넘어가세요.

---

## 2단계 — 매주 쓸 수 있게 정리

아래 내용을 붙여넣으세요.

```
이 작업을 매주 반복할 예정이야.
폴더를 반복 업무에 맞게 정리해줘.
지금까지 사용한 요청 내용도 함께 저장해줘.
```

**완료 확인:** 폴더 안에 하위 폴더들이 새로 생겼으면 3단계로 넘어가세요.

---

## 3단계 — 분석 방식 저장

아래 내용을 붙여넣으세요.

```
방금 재고 데이터를 분석한 방식을 저장해둬.
다음 주에 같은 방식으로 바로 사용할 수 있게 weekly-inventory-report라는 이름으로 만들어줘.
```

**완료 확인:** "저장됐습니다" 또는 파일이 생성됐다는 메시지가 나오면 4단계로 넘어가세요.

---

## 4단계 — 다음 주 데이터에 적용

**먼저 파일을 복사해주세요.**
`추가데이터/` 폴더에 있는 CSV 파일을 복사해서 `데이터/` 폴더에 붙여넣으세요.

그런 다음 아래 내용을 붙여넣으세요.

```
/weekly-inventory-report

새로 추가된 재고 데이터 파일을 같은 방식으로 분석해서 이번 주 리포트를 작성해줘.
```

**완료 확인:** 이번 주 리포트 파일이 새로 생겼으면 완료입니다.

---

## 실습 완료 후

생성된 리포트를 열어보세요. 긴급 항목이 자동으로 분류되어 있으면 성공입니다.

막히는 부분이 있으면 강사에게 문의하세요.
