# 마케팅·운영팀 실습 가이드

이 폴더에는 회의록 파일 2개가 있습니다. 아래 순서대로 진행하세요.

---

## 1단계 — 4월 회의록 정리

아래 내용을 Claude Code에 붙여넣으세요.

```
이 폴더에 있는 파일들을 모두 읽어줘.
4월 회의 내용을 찾아서 아래 형식으로 정리하고 저장해줘.

형식:
1. 회의 개요 (날짜, 팀, 참석 추정)
2. 마케팅 현황 (캠페인 성과, 채널별 분석, 예산)
3. 운영 현황 (물류·배송, 품질, 재고)
4. 결정 사항
5. 액션 아이템 (담당자, 기한)
6. 리스크 및 확인 필요 사항
7. 다음 달 계획
8. 한 줄 요약

- 발화자가 불분명하면 "담당 추정"으로 표시해줘
- 확정되지 않은 내용은 "확인 필요"로 표시해줘
- 실무자가 바로 공유할 수 있는 수준으로 정리해줘
```

새로운 문서를 만들 때 "HTML 파일로 작성해줘"라고 요청하면, 크롬 브라우저에서 바로 열어볼 수 있는 형태의 파일로 저장됩니다.

**완료 확인:** 회의록이 정리된 파일이 새로 생겼으면 2단계로 넘어가세요.

---

## 2단계 — 매달 쓸 수 있게 정리

아래 내용을 붙여넣으세요.

```
이 작업을 매달 반복할 예정이야.
폴더를 반복 업무에 맞게 정리해줘.
지금까지 사용한 요청 내용도 함께 저장해줘.
```

**완료 확인:** 폴더 안에 하위 폴더들이 새로 생겼으면 3단계로 넘어가세요.

---

## 3단계 — 정리 방식 저장

아래 내용을 붙여넣으세요.

```
4월 회의록을 정리한 방식을 저장해둬.
다음 달에 같은 방식으로 바로 사용할 수 있게 monthly-meeting-summary라는 이름으로 만들어줘.
```

**완료 확인:** "저장됐습니다" 또는 파일이 생성됐다는 메시지가 나오면 4단계로 넘어가세요.

---

## 4단계 — 5월 회의록 정리

아래 내용을 붙여넣으세요.

```
/monthly-meeting-summary

5월 회의록을 4월과 같은 방식으로 정리해줘.
```

**완료 확인:** 5월 회의록 파일이 새로 생겼으면 완료입니다.

---

## 실습 완료 후

4월 파일과 5월 파일을 나란히 열어보세요. 같은 형식으로 정리되어 있으면 성공입니다.

막히는 부분이 있으면 강사에게 문의하세요.
